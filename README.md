# Bank Management Application

A **desktop banking application** built with Java Swing and MySQL. Manage customer accounts, deposit/withdraw money, and track transactions through a simple GUI.

## Features
- Create, edit, and delete customer accounts
- Deposit and withdraw funds
- View account and transaction history
- User-friendly interface built with Java Swing
- Secure storage with MySQL database

## Tech Stack
- Frontend: Java Swing
- Backend/Database: MySQL
- Programming Language: Java
